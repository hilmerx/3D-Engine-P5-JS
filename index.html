HILMERX-3D - A 3D engine for the browser.

You're probably looking at it right now.

I noticed that the library P5s 3D engine was broken, nothing was working like it should, so I built a new one from scratch using javascript and the canvas HTML object. It may be simple and can only do very basic stuff at the moment, but it does the work I want it to.
